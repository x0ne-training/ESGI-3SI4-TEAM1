# Reigns

**Reigns** est un petit jeu en C où vous incarnez un monarque et devez prendre des décisions pour gouverner votre royaume. Chaque choix a un impact sur vos ressources et votre royaume.

---

## Fonctionnalités

- Gestion simple du royaume avec différentes statistiques.
- Prise de décisions avec conséquences sur la population, l’armée, le trésor et l’église.
- Interface en terminal minimaliste mais intuitive.
- Sauvegarde des progrès via les commits dans le code (pour développeurs).

---

## Prérequis

- GCC (ou tout autre compilateur C compatible C11)
- Make

Sur Arch Linux :

```bash
make
./reigns
