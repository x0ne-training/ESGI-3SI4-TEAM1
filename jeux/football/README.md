# Mini FIFA CLI

Mini FIFA CLI est un petit jeu de football en ligne de commande inspiré de FIFA. Vous incarnez un joueur et prenez des décisions qui affectent ses statistiques et son équipe.

## Fonctionnalités

- Choix interactifs influençant les statistiques du joueur
- Suivi des stats : attaque, défense, endurance et technique
- Interface simple en terminal
- Compilation facile avec Makefile

## Installation

1. Cloner le dépôt :
```bash
git clone <url-du-depot>
cd <nom-du-repo>
```
2. Compiler le jeu avec make:
```bash
make
```
3. Lancer le jeu
```bash
./mini_fifa
```
