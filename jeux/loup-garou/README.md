# Loup-Garou CLI

Un jeu de type Loup-Garou en ligne de commande.

## Règles

- Le village contient plusieurs joueurs : villageois, loups-garous et rôles spéciaux.
- Chaque tour représente une **nuit** (loups attaquent) et un **jour** (vote du village).
- Le jeu continue jusqu'à ce que tous les loups ou tous les villageois soient éliminés.

## Compilation

```bash
make
./loup_garou
```
