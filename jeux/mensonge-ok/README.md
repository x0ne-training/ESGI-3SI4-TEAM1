# Jeu du Mensonge

## Description
Un petit jeu en C simple et drôle.  
Règle : le jeu pose des questions absurdes. Tu dois répondre **le contraire de la vérité**.  
Si tu dis la vérité → tu perds.  
Si tu mens correctement → tu continues.

---

## Fichiers

- `main.c` : point d'entrée du jeu
- `functions.c` et `functions.h` : fonctions pour poser les questions et vérifier les réponses
- `Makefile` : pour compiler facilement
- `README.md` : ce fichier

---

## Compilation & Exécution

Dans le terminal, depuis le dossier du projet :

```bash
make        # compile le jeu
./mensonge  # lance le jeu
