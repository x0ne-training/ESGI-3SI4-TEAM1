# La Forêt Maudite

Un mini jeu en C où vous devez survivre dans une forêt remplie de dangers.

## Règles

- Chaque jour, vous pouvez :
    - Explorer (risque de blessure, gain possible de nourriture)
    - Chasser (gain de nourriture possible)
    - Vous reposer (régénère énergie et santé)
- Vous consommez **1 unité de nourriture par jour**.
- Le jeu se termine si votre **santé** ou votre **énergie** atteint 0.
